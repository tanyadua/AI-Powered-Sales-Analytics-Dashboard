# AI-Powered Sales Analytics Dashboard

## Overview

This project analyzes sales performance across products, categories, and regions using Microsoft Excel.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Data Visualization
* Claude AI

## Dashboard Features

* Total Sales KPI
* Total Profit KPI
* Sales by Product Analysis
* Sales by Region Analysis
* Sales by Category Analysis
* AI-Generated Business Insights

## Key Results

* Total Sales: ₹8,77,500
* Total Profit: ₹2,23,700
* Electronics emerged as the highest-performing category.
* Laptops generated the highest revenue.
* South region delivered the strongest sales performance.
